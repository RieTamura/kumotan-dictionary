# kumotan-dictionary

くもたん (Kumotan) アプリ用の辞書データリポジトリです。

## ファイル

| ファイル | 説明 |
|----------|------|
| `jmdict.db.gz` | JMdict辞書データ（SQLite、Gzip圧縮） |
| `metadata.json` | バージョン情報 |

## ダウンロードURL

```
https://rietamura.github.io/kumotan-dictionary/jmdict.db.gz
https://rietamura.github.io/kumotan-dictionary/metadata.json
```

## ライセンス

辞書データは [JMdict](https://www.edrdg.org/wiki/index.php/JMdict-EDICT_Dictionary_Project) を使用しています。
